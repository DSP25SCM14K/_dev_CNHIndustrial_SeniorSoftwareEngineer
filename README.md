Dark, animated portfolio site for Dev Kumar, tailored around cloud telemetry services, event-driven .NET systems, Azure ingestion, versioned data contracts, APIs, observability, and engineering mentorship.

## Research inputs

- Public senior backend/API engineering role signals: cloud-based applications, consistent enterprise data, Domain-Driven Design, sequence diagrams, flowcharts, code reviews, mentorship, and cross-platform integration.
- Public senior software role signals: .NET Core, Entity Framework, SQL, Azure/AWS, REST/SOAP APIs, debugging, Terraform, Agile, precision agriculture, IoT, CI/CD, containers, and Linux systems.
- Azure Event Hubs documentation: managed real-time event ingestion at large scale, Apache Kafka compatibility, and telemetry ingestion patterns.
- Azure Service Bus documentation: dead-letter queues for messages that cannot be delivered or processed.
- Azure Application Insights documentation: distributed tracing, telemetry, and application performance monitoring.
